* This project will use NASA's Open MCT frame work 

* The first phase of this project will 
    * `replay NASA turbofan telemetry through Open MCT and show anomaly alerts`

* so the results will be transferred into the Open MCT frame to simulate a real world environment



## Main Goal

Build a mission-control-style system that replays \
aerospace engine telemetry, monitors sensor health, detects anomalies, predicts remaining useful life, \
and displays alerts in Open MCT.
